# CRM
A simple CRM that covers the following use-cases:

- Clients:
  1. Browse services
  2. View example projects
  3. View contacts
  4. Place orders

- Managers:
  1. View clients
  2. View orders
  3. View analytics
  4. Create orders
  5. Manage order state and details
